Do This Next

This application is designed to support those of us who are not satisfied with the state of "TO DO" list offerings currently in the public.

Input a task name in the fields below. The only mandatory field is the task name. Obviously the more of the fields you populate, the better equipped you will be to track this task. More as we go... 

The following fields are defined in the data model:

name
priority
due_date
description
tags
creator
owner
status

Only a few of them are gathered by the input form (for now)
